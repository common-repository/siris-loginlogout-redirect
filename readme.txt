Manage your users login and logout experience with this easy to use plugin.

== Description ==

Siris Login Logout Redirect points users to any url you choose when logging in to your site or when they logout of your site. The plugin also lets you choose a static login page from your site to take the users to if they face login issues.

The plugin also allows you to hide the Admin bar on the external facing part of your site.

== Installation ==
1. Download and unzip the plugin.
2. Upload 'siris-login-redirect' folder to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
1. What if one of the options is left empty?
The plugin will use the Wordpress standard option if the plugin option is left empty.
